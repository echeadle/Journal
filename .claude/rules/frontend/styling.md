# Styling Rules

- One CSS file per component
- BEM naming: block__element--modifier
- CSS variables defined in theme.js
- Color palette: Sunshine #FFB347, Sky #87CEEB, Mint #98D8C8, Cream #FFF8F0
- Mobile-first: breakpoints at 480px, 768px, 1024px
- Minimum 44px tap targets for touch
- Rounded corners, soft shadows, smooth transitions
- Responsive design principles for all screen sizes
